Bibliotecas/frameworks

  • Bootstrap

    Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript.

  • Tailwind CSS

    Tailwind CSS é uma estrutura CSS de código aberto. A principal característica desta biblioteca é que, ao contrário de outros frameworks CSS como o Bootstrap, ela não fornece uma série de classes predefinidas para elementos como botões ou tabelas.

  • SASS

    O SASS é uma linguagem de extensão do CSS, a sigla significa “Syntactically Awesome Style Sheets”. A sua ideia é adicionar recursos especiais como variáveis, mixins, funções e operações e outras opções

  • React JS

    O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

  • Font Awesome

    Font Awesome é um conjunto de ferramentas de fontes e ícones com base em CSS e LESS.

  • Typescript

    TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem.
